# [Nazwa łatki]
Opis w jednym zdaniu.

## V-QRW

Opisz krótko swoją motywację, czym się zdenerwowałe(a)ś?

## Cel

Czego spodziewasz się po zmianie

## Metoda

_kursywą kroki opcjonalne_

1. Sadzić
   1. _Marchew_
   1. _Pietruszkę_
2. Zebrać
3. Umyć
4. Ugotować
5. Zjeść
6. Powtarzaj punkty 1-5, aż się najesz

## Tips & Tricks

* Sadź w ziemi, nie w piachu
* Obierz zanim zjesz
